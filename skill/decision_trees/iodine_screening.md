# Decision Tree — Iodine Screening & Supplementation

## 主流程（每位孕婦初診都該跑）

```
START: 任何孕婦（preconception 更佳）
│
├── 三題篩查
│   1. 「家裡用的是加碘鹽嗎？」
│   2. 「孕婦維他命有含 iodine 嗎？」
│   3. 「一週吃幾次海帶 / 海苔？」
│
├── 任一答 NO 或不確定 → 啟動 iodine multivitamin
│
└── 全部 YES + 海藻 ≥ 1×/週 → 評估是否過量（>500 µg/d）
```

## 標準補充建議
- **150–200 µg KI/day**（含於 prenatal multivitamin）
- 持續：preconception → 整孕期 → lactation 6 個月（or 至斷奶）
- Thyroidectomy / RAI 病人 **同樣需要**（給胎兒 + 哺乳，非給母體腺體）
- 哺乳期 breastmilk iodine 反映近期攝取

## 為何不檢驗 spot UIC（Case 7 選項 3 的陷阱）
- 個體內 CV 30–40%
- WHO/IGN 規範 ≥ 125 個 spots 才能評族群 median
- 孕期 creatinine 校正反而引入偏誤（GFR + 肌肉量變化）
- → 個別孕婦判定靠 **飲食行為史**，不是 spot UIC

## 食物 iodine 速查
| 來源 | 含量 | 可靠度 |
|---|---|---|
| **加碘鹽** | 20–40 µg/g | 高 |
| 海鹽 / 玫瑰鹽 / 喜馬拉雅鹽 | < 2 µg/g | — |
| 海帶 / 紫菜（乾重） | 16–8000 µg/g（500–1000× 變異） | 低 |
| 海藻錠 | FDA 抽樣標示誤差 11× | 低 |
| 雞蛋 | ~25 µg/顆 | 中 |
| 月子海帶湯（單餐） | 1000–5000 µg | 過量風險 |

## 過量 iodine 評估
- UL 500–600 µg/day
- 過量風險族群：
  - 月子文化海帶 / 紫菜湯（特別是日韓家庭 miyeokguk）
  - 海藻錠（標示不可信）
  - Povidone-iodine 暴露：剖腹產皮膚消毒、新生兒臍帶照護、會陰消毒
  - CT / contrast（單次 ~35 g）
  - Amiodarone

## TPOAb (+) 補碘特別注意
- 突然 > 500 µg/d 可能誘發 Wolff-Chaikoff escape failure 與 thyroiditis flare
- **僅** 標準 150–200 µg/d，不額外鼓勵海藻

## Graves + ATD 補碘
- 不適用 universal 250 µg/d
- 個別化評估，避免高劑量海藻
- 加碘可能改變 ATD 劑量需求

## 病人衛教（Health Belief Model 對應）
- **Perceived susceptibility**: 「即使 TFT 正常，台灣孕婦中位 UIC 148，剛好在 WHO 建議下緣」
- **Perceived severity**: 「碘缺乏可能影響寶寶腦部發展（IQ 約 3–4 分）」
- **Perceived benefits**: 「含碘維他命已被證實能拉抬尿碘 +41 µg/L」
- **Perceived barriers**: 「一天一顆，多數品牌都不貴；要看成分標示有 iodine 或 KI」
- **Norm holders**: 與婆婆 / 月嫂 / 月子中心 / 育兒社團衛教（單靠 5 分鐘診間衛教不夠）

## 診間問答常見問題
1. 「我吃海帶湯了還需要補嗎？」→ 是，海帶含量 16–8000 µg/g 不可靠
2. 「哺乳期繼續補嗎？」→ 是，breastmilk iodine 反映近期攝取
3. 「我吃孕婦綜合維他命了，應該夠了吧？」→ **不一定**，台灣多數品牌不含碘或 < 50 µg；要查標示
4. 「玫瑰鹽 / 海鹽不是更天然嗎？」→ iodine 含量極低 (< 2 µg/g)，不能取代加碘鹽
5. 「會不會吃太多反而不好？」→ 標準劑量 150–200 µg/d 安全；UL 500–600；避免大量海藻錠

## ⚠️ Hard Stops
- 已知 amiodarone 暴露 / contrast 大量 → 不建議再補碘，個別化評估
- TPOAb 強陽 + 已 high-dose iodine 史 → 警覺 Wolff-Chaikoff escape failure
- 已知 Graves on ATD → 不適用 universal 250 µg/d
- 嚴重 TPOAb (+) thyroiditis 急性 → 暫緩補碘
