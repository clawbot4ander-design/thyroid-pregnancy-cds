# Iodine in Pregnancy

## 孕期碘需求 ↑ ~50%
- 母體 metabolic rate ↑ 15–20%
- 甲狀腺激素生產 ↑ 50%
- 腎碘清除 ↑ 30–50%（GFR↑）
- 胎盤主動運輸 50–75 µg/day → 胎兒
- 胎兒 GA 12 週後依賴自身合成 → 需要碘原料

## 建議攝取量
- **WHO**：250 µg/day（孕期 + 哺乳期）
- **台灣 DRI 第八版**：225 µg/day
- **Tolerable Upper Limit (UL)**：500–600 µg/day

## 台灣 iodine 流行病學
- **NAHSIT-PW 2017–2019**: 孕婦 median UIC 148–156 µg/L（WHO 充足下限為 150）
- **Pan LH et al. BMC Pregnancy and Childbirth 2025**: T1 / T2 / T3 median 156.3 / 146.2 / 169.9 µg/L
- 多數市售台灣 prenatal vitamin **不含碘** 或含量 < 50 µg
- **含碘 multivitamin → UIC +41 µg/L**（NAHSIT 觀察）

## 政策史
- 1967 年起強制食鹽加碘
- 2003 年 WTO 自由化後改為「自願標示」
- NAHSIT 監測 UIC 持續下滑至 borderline insufficient
- 缺乏 impact evaluation 的 policy reversal 案例

## 食物來源 iodine 含量
| 來源 | 含量 |
|---|---|
| 加碘鹽 | 20–40 µg/g |
| 海鹽 / 玫瑰鹽 / 喜馬拉雅鹽 | < 2 µg/g |
| 海帶 / 紫菜（乾重） | 16–8000 µg/g（500–1000× 變異） |
| 海藻錠 | FDA 抽樣標示誤差可達 11× |
| 月子海帶湯（單餐） | 1000–5000 µg（過量風險） |
| 乳製品（西方主要 hidden source） | 在台灣攝取量低 |
| 雞蛋（含碘飼料） | ~25 µg/顆 |

## Spot UIC 解讀陷阱（重要）
- **個體內 CV 30–40%** → 不能用於個別孕婦診斷
- WHO/IGN 規範至少 ≥ 125 個 spots 才能評族群 median
- **孕期 creatinine 校正反而引入偏誤**（GFR + 肌肉量變化）
- → 個別孕婦判定需要的是 **飲食行為史**，不是 UIC

## 個別孕婦 iodine 篩查三題
1. 「家裡用的是加碘鹽嗎？」
2. 「孕婦維他命有含 iodine 嗎？」
3. 「一週吃幾次海帶 / 海苔？」

## 碘補充劑量建議
- 起始 150–200 µg KI/day（含於 multivitamin）
- 持續：preconception → lactation 6 個月 / 斷奶
- Thyroidectomy / RAI 病人也需補（不是給腺體，是給胎兒 + 哺乳）

## 過量碘風險族群
- 月子海帶湯（麻油海帶 / 紫菜湯，單餐 1000–5000 µg）
- 韓裔 / 日裔家庭的 miyeokguk（產後常見 UIC > 1000 µg/L）
- 海藻錠保健品（標示不可信）
- Povidone-iodine 暴露：剖腹產皮膚消毒、新生兒臍帶照護、會陰消毒（哺乳期可轉移）
- CT / 血管攝影 iodinated contrast（單次 ~35 g）
- Amiodarone

## TPOAb (+) 補碘特別注意
- 突然 > 500 µg/d 可能誘發 Wolff-Chaikoff escape failure 與 thyroiditis flare
- 建議 150–200 µg/d 即可，避高劑量海藻補充

## Graves + ATD 補碘
- 不適用 universal 250 µg/d
- 加碘可能改變 ATD 劑量需求 + 影響胎兒甲狀腺 Wolff-Chaikoff 反應
- 個別化評估，避高劑量海藻

## 碘缺乏與 child IQ（observational）
- **Bath et al. Lancet 2013** ALSPAC: UIC < 150 µg/g Cr → 子女 8–9 歲 verbal IQ、reading 顯著較低
- **Murcia INMA 2018**: 母體 iodine intake 不足 ↔ 1 歲 psychomotor scale 較低
- **Abel et al. JN 2017** Norwegian MoBa: iodine intake < 160 µg/day ↔ child language delay、ADHD 風險上升
- Effect size: child verbal IQ ↓ 3–4 分

## 碘介入 RCT 限制
- **Gowachirapant 2017 Lancet D&E**（Thailand/India）：mild ID 區補碘對 5 歲 IQ 無顯著
- 介入時機晚（多孕中期）+ 追蹤短
- **缺乏 preconception–lactation RCT**
