# Pregnancy Thyroid Physiology

## hCG–TSH 軸
- hCG GA 8–11 週達高峰，可達 50,000–100,000 IU/L
- α subunit 與 TSH 同源，弱結合 TSH receptor → 刺激甲狀腺
- 結果：fT4 ↑、TSH ↓（生理性）
- Hyperglycosylated hCG isoforms（HG / multiple gestation / molar）對 TSHR 親和性更強 → GTT 風險 ↑

## Estrogen–TBG 軸
- Estrogen 減少 TBG sialylation → TBG 半衰期延長
- TBG 在 GA 6–8 週開始上升，20 週 plateau
- Total T4 / T3 ↑ 約 1.5×（free T4 受 TBG 影響較小）

## Trimester-specific TSH reference（pregnancy-specific）
- 早孕期 TSH 上限 ↓ ~0.5 mU/L、下限 ↓ ~0.4 mU/L
- 一般使用 0.1–4.0 mIU/L 為早孕期 TSH range（vs 非孕 0.5–4.5）
- **必須採 population-specific, assay-specific, trimester-specific reference**
- 16 週後改用 Total T4 × 1.5× 非孕上限 為更穩定 metric

## Maternal thyroid 適應
- 甲狀腺體積 ↑ 10–15%（碘充足區常被忽略，台灣 borderline 區更明顯）
- T4/T3 production ↑ 50%
- 腎 GFR ↑ 30–50% → iodine clearance ↑
- 額外需要 50–100 µg iodine/day

## Placental / fetal 機轉
- **Placental Type 3 deiodinase (D3)** 將 T4/T3 不可逆代謝為 rT3 / T2 → 胎盤屏障，緩衝胎兒過度暴露
- **Fetal brain D2** 將 T4 局部去碘成 T3（cortex 內 T3 不來自 maternal T3）
- **Substances crossing placenta**: 大多 cross；**TSH does NOT cross**
- Thionamides (PTU/MMI/CMZ) 自由通過胎盤
- LT4 通過受 D3 與 placental transporter 限制 → 因此 block-and-replace 是禁忌
- TRAb (IgG) 通過胎盤；TPOAb (IgG) 也通過但**非 thyroid-stimulating**

## Fetal thyroid 發育時序
- GA 10–12 週：NIS (sodium-iodide symporter) 表現開始 trapping
- GA 12 週後：開始 organification + T4 合成
- GA 12 週前：胎兒 CNS **完全依賴** maternal T4 跨胎盤
- GA 20 週：胎兒甲狀腺完全功能
- Cortical neurogenesis 集中於 8–18 週 → 神經發育 critical window
- Wolff-Chaikoff escape 機制要 GA 36 週後才成熟 → 新生兒對碘負荷敏感

## GTT vs Graves 鑑別重點
| 項目 | GTT | Graves |
|---|---|---|
| Etiology | hCG-mediated | autoimmune (TRAb) |
| Prevalence | 1–3%（HG 30–60%） | 0.1–0.4% |
| History | 無甲狀腺病史 | 多有 |
| Goiter | 通常無 | 多有 diffuse |
| Ophthalmopathy | 無 | 25–50% |
| TT3/TT4 ratio (ng/μg) | < 20 | > 20 偏向 |
| TRAb | 陰性 | 陽性 |
| Course | 14–20 週自緩 | 持續 |
| Therapy | 支持治療 | ATD ± others |
